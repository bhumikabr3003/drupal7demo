Please follow the instruction to use this module
1. Go to Module page and enable the module "Current posts"
2. Go to Structure >> Block >> look for block "Current posts" in Disable section
2. Now, select "Content" for the Region Dropdown option and press "Save blocks" button
3. On the homepage you will see the block with title "My latest 5 articles" which will show 5 latest articles
NOTE: You must have few articles added in your Drupal site
NOTE: Block "Current posts" has to be added in region
